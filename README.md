# Simon Game

A simple Simon game built using HTML, CSS, and JavaScript (jQuery). This project is a fun way to practice front-end web development skills.

## Introduction

The Simon game is a memory game where players must repeat a sequence of colors in the correct order. Each round, a new color is added to the sequence, increasing the difficulty.

## Features

- Press any key to start the game.
- Visual and audio feedback for user interactions.
- Automatically checks user input against the game pattern.
- Displays a "Game Over" message when the user makes a mistake.

## Usage

Press any key to start the game. The game will generate a sequence of colors which you need to repeat by clicking the colored buttons. The sequence gets longer with each round. If you make a mistake, the game will display "Game Over" and you can restart by pressing any key.

## Gameplay

1. Press any key to start.
2. Watch the sequence of colors.
3. Repeat the sequence by clicking the buttons in the same order.
4. Each round, a new color will be added to the sequence.
5. The game ends when you click the wrong color.
